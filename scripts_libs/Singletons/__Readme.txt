=========================================================================
Singleton 
=========================================================================

Singleton - паттерн проектирования
- он же уникальный объект на сцене, сколько бы объектов ни создавали, он такой один 
  (таковы условия)

- Создаем объект Level на сцене, добавляем на него скрипт Level (создающий из этого объекта Singleton)
- Объект Level - является Singleton 
- Дальше мы можем обрщаться к этому объекту как к объекту с настройками через его поле Instance 

  Level.Instance.NumberOfBalls; 

- Это нужно также для оптимизации, чтобы не искать объект по Тэгу или Типу или если объект не удается 
  привязать сразу на сцене, например если объекты, которые хотят его использовать создаются динамически 
  (т.е. во время выполнения программы)
