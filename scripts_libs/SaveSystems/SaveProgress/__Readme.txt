=========================================================================
Система сохранения прогресса игры в файл при переходе между уровнями
=========================================================================

- GameManager.cs 
  - Разместить на объект GameManager в сцене
  - Содержит методы для переходов от сцены к сцене и методя для обработки победы и поражения,
    которые вызываются в других скриптах 
  - Содержит методы для сохранения и загрузки данных игры SaveData, LoadData

- Progress.cs
  - Разместить на объект Progress в сцене ( это сквозной объект Singleton c методом DontDestroyOnLoad(gameObject); в Awake 
  - В нем хранятся данные общие от уровня к уровню, которые необходимо сохранять 

- ProgressData.cs
  - Простой класс повторяющий поля объекта Progress, необходим для сохранения данных
  - Используется в качестве посредника в системе сохранения 
  - Не является наследником Monobehaviour  (методы Unity в нем не работают!)
  - Зависит от объекта Progress

- SaveSystem.cs
  - Статический С# класс 
  - Основной класс, реализующий сохранение прогресса игры в файл
  - Не является наследником Monobehaviour  (методы Unity в нем не работают!)
  - Зависит от объектов Progress и ProgressData
